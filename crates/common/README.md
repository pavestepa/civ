# civ-common

Shared foundation types for the Civ project.

## Responsibility

- Player, unit, city, and entity identifiers
- Application state enums
- Save format version constants
- Cross-layer error types

## Dependencies

None of Bevy, React, Tauri, or QuickJS. Safe to use from every crate.

## Future

- Replay header types
- Network protocol version markers
- Platform capability flags
